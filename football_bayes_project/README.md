# Football Bayesian Model Project

## Where to put files
- Put your ODS dataset in `data/` and name it `dataset.ods`
- Put your Python model code in `src/` and name it `bayes_model.py`

## Suggested run command
From the project root:

```bash
python src/bayes_model.py
```

## Suggested outputs
Save model outputs into the `outputs/` folder.
